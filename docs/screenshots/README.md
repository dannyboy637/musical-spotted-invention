# Screenshots

This directory contains screenshots for the README and documentation.

## Required Screenshots

Take these screenshots and save them here:

1. **dashboard.png** - Executive Dashboard showing KPIs, revenue chart, top/bottom items
2. **menu-engineering.png** - Menu Engineering page with BCG scatter plot
3. **time-intelligence.png** - Time Intelligence page with heatmap
4. **reports.png** - Reports page showing report list and preview

## How to Take Screenshots

1. Run the app locally with sample data
2. Use browser DevTools to set viewport to 1440x900 for consistency
3. Take full-page screenshots using browser extensions or DevTools
4. Optimize images using tools like ImageOptim or TinyPNG
5. Target file size: < 500KB each

## Recommended Tools

- **Chrome DevTools**: Device toolbar for consistent viewport
- **Full Page Screen Capture**: Chrome extension for full-page shots
- **ImageOptim**: Mac app for lossless compression
- **TinyPNG**: Online tool for PNG/JPEG compression
